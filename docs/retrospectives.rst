Retrospectives
==============

This page provides links to all retrospectives conducted by the team.  Each retrospective
captures meeting notes, action items, and process improvements.  Retrospectives are stored
in the :code:`docs/retros` directory.

Retrospectives are listed in reverse chronological order (newest first).

All Retrospectives
------------------

.. toctree::
   :maxdepth: 1
   :glob:

   retros/*

.. note::
   When creating a new retrospective, add a new :code:`.rst` file in the :code:`docs/retros`
   directory following the naming convention :code:`YYYY-MM-DD-retrospective.rst`.  The file
   will be automatically included in the list above.
